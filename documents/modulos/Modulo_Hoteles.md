# Módulo: Gestión de Hoteles — Guía para Clientes

Este módulo permite **configurar y administrar la información base** de cada hotel en AlojaSys.

## ¿Qué podés configurar?

- **Datos del hotel**: nombre, dirección, teléfono, email.
- **Información legal**: razón social, CUIT/CUIL (base para facturación).
- **Ubicación**: país, provincia, ciudad.
- **Horarios**: hora de check-in y check-out.
- **Zona horaria**: para que el sistema calcule fechas/horas correctamente.
- **Automatizaciones**:
  - **Auto check-in** (opcional).
  - **Auto check-out** (habilitado por defecto).
  - **Auto no-show** (opcional).

## Check-out automático (lo más importante)

El sistema puede hacer **check-out automáticamente** cuando pasa la fecha/hora de salida del hotel:

- **Si la fecha de check-out ya pasó**: procesa el check-out inmediatamente.
- **Si es el día de check-out**: espera hasta la hora configurada (ej: 11:00).
- Luego **libera la habitación** para nuevas reservas.

## Beneficios

- **Información centralizada** por hotel.
- **Menos trabajo manual** con automatizaciones.
- **Soporte multi-hotel** (una empresa puede tener varios hoteles).
