# Módulo: Gestión de Empresas — Guía para Clientes

Este módulo permite administrar una **empresa/grupo** que puede tener **múltiples hoteles**.

## Qué permite

- Agrupar hoteles bajo una misma empresa.
- Definir **configuraciones globales** (según el caso), por ejemplo:
  - políticas de pago estándar
  - métodos de pago habilitados
  - usuarios con acceso multi-hotel
- Acceder a **reportes consolidados** del grupo.

## Beneficios

- Gestión centralizada para cadenas o grupos.
- Consistencia de configuración entre hoteles.
- Escalabilidad cuando el negocio crece.
