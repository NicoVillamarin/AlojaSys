# Módulo: Housekeeping (Gestión de Limpieza) — Guía para Clientes

Este módulo permite gestionar tareas de limpieza y mantenimiento con **seguimiento, asignación** y estados claros por habitación.

## Activación (según plan)

- Es un módulo **opcional** según el plan contratado.
- Si no está activo, igual podés usar el **estado de limpieza** como etiqueta manual.

## Estados de habitación (importante)

Cada habitación tiene:

- **Estado principal**: disponible / ocupada / mantenimiento / etc.
- **Estado de limpieza (subestado)**: **para limpiar** / **en limpieza** / **limpia**

Regla clave: al hacer **check-out**, la habitación queda en **“Para limpiar”**.

## Si el módulo está activo

Se habilita un flujo completo:

- tareas de salida (checkout)
- tareas diarias
- tareas de mantenimiento
- (opcional) checklists
- asignación de personal y zonas

## Beneficios

- Orden y visibilidad del trabajo diario.
- Menos errores (habitaciones listas a tiempo).
- Mejor coordinación entre recepción y limpieza.
