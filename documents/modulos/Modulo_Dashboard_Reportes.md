# Módulo: Dashboard y Reportes — Guía para Clientes

El dashboard muestra **métricas clave del hotel** en tiempo real, con reportes para análisis.

## Qué métricas vas a ver (ejemplos)

- **Habitaciones**: total, disponibles, ocupadas, mantenimiento.
- **Reservas**: pendientes, confirmadas, canceladas, check-ins/check-outs del día.
- **Huéspedes**: esperados, alojados, saliendo.
- **Finanzas**: ingresos del día, tarifa promedio, ocupación.

## Reportes automáticos

- Diarios / semanales / mensuales.
- Por hotel (y, si corresponde, consolidados por empresa).

## Beneficios

- Visión rápida para tomar decisiones.
- Seguimiento de ocupación e ingresos.
- Base para comparar períodos (histórico).
