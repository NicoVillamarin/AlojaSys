# Módulo: Gestión de Usuarios — Guía para Clientes

Este módulo administra **quién accede** al sistema y **qué puede hacer** cada persona.

## Qué se configura

- **Usuarios** (personal).
- **Roles/perfiles** (ej: administrador, gerente, recepcionista, contador).
- **Hoteles asignados** (un usuario puede tener 1 o varios hoteles).

## Beneficios

- Seguridad: cada uno ve y usa lo que corresponde.
- Control operativo: permisos acordes a cada rol.
- Soporte multi-hotel para personal compartido.
