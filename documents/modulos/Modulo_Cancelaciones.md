# Módulo: Políticas de Cancelación — Guía para Clientes

Este módulo define **cuándo se puede cancelar una reserva** y qué **penalidades** aplican.

## Regla clave (importante)

Las políticas se calculan **en base al tiempo restante hasta el check-in** (no desde la fecha de creación de la reserva).

## Cómo se configura (idea)

Normalmente se define una escala:

- **Cancelación gratuita** (hasta X horas/días antes)
- **Cancelación parcial** (penalidad, hasta Y horas/días antes)
- **Sin cancelación** (luego de ese punto)

Ejemplo:

- Gratis: hasta 72h antes
- Parcial: entre 24h y 72h antes (ej: penalidad 50%)
- Sin cancelación: menos de 24h antes

## Beneficios

- Reglas claras para el hotel y para el huésped.
- Menos conflictos (criterios consistentes).
- Se integra con devoluciones, vouchers y reembolsos.
