# Módulo: Comprobantes de Señas y Devoluciones — Guía para Clientes

Este módulo genera y organiza **comprobantes (PDF)** para:

- **Señas** (pagos parciales)
- **Devoluciones** (reembolsos)

## Cómo funciona

- Cuando se registra una **seña**, el sistema puede:
  - asignar un **número de comprobante**
  - generar un **PDF**
  - dejarlo disponible para ver/descargar
- Cuando se confirma un **reembolso**, el sistema puede:
  - generar el **comprobante de devolución** (PDF)
  - guardar una URL permanente

## Qué vas a ver en la interfaz

- Listas con búsqueda y filtros (por hotel, huésped, fecha, método, estado).
- Acciones rápidas: **ver / descargar**.

## Beneficios

- Orden administrativo (todo centralizado).
- PDFs consistentes para entregar al huésped.
- Integración natural con pagos, devoluciones y facturación (si está habilitada).
