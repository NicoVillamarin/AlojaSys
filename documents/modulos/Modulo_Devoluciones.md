# Módulo: Políticas de Devolución — Guía para Clientes

Este módulo define **cómo se devuelve el dinero** (o se compensa) cuando corresponde reembolso.

## Qué podés configurar

- **Tiempos/condiciones** de devolución (completa / parcial / sin devolución).
- **Métodos**:
  - método de pago original
  - transferencia
  - efectivo (manual)
  - **voucher** (crédito para futuras reservas)
- **Mensajes** para comunicar al huésped (recomendado).

## Flujo típico

- Se cancela la reserva.
- El sistema evalúa política (cancelación) + calcula penalidad.
- Aplica política de devolución y define el monto final a devolver/compensar.

## Beneficios

- Proceso consistente y auditable.
- Mejor experiencia del huésped (reglas claras).
- Base para automatizaciones de reembolso.
