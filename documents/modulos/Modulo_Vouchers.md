# Módulo: Vouchers de Crédito — Guía para Clientes

Este módulo permite generar **vouchers de crédito** como alternativa a devolver dinero, ayudando a **retener clientes** y mejorar el flujo de caja.

## Cómo funciona (simple)

Cuando un cliente cancela y corresponde reembolso:

- Opción A: **reembolso** al método de pago (según política).
- Opción B: **voucher** con un código único (ej: `VCH-2025-00045`).

## Qué incluye un voucher

- **Código único**
- **Monto**
- **Vencimiento** (configurable)
- **Estado**: activo / usado / expirado / cancelado

## Cómo se usa

- En una nueva reserva, el cliente ingresa el **código**.
- El sistema valida (activo, no expirado) y aplica el **descuento** al total.

## Beneficios

- Para el hotel: menos salida inmediata de dinero + fidelización.
- Para el cliente: crédito claro, usable en futuras reservas.
