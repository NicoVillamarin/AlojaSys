# Módulo: Gestión de Tarifas — Guía para Clientes

Este módulo permite configurar **precios dinámicos**, promociones e impuestos de forma flexible.

## Qué podés configurar

- **Planes de tarifa**: por ejemplo “Tarifa Estándar”.
- **Reglas**: por fechas/días (ej: fin de semana), por habitación o tipo.
- **Promociones**: códigos de descuento, condiciones y vigencia.
- **Impuestos**: porcentajes o montos aplicables.

## Cómo se calcula un precio (idea simple)

Para una cotización, el sistema combina:

- precio base + reglas aplicables + impuestos – descuentos

## Restricciones de venta

- **CTA**: cerrado a llegadas.
- **CTD**: cerrado a salidas.
- **Días cerrados**.
- **Estadía mínima/máxima**.

## Recomendación práctica

- Usá tarifas para **venta directa** en AlojaSys.
- Si trabajás OTAs con channel manager (ej: Smoobu), definir una estrategia clara de “quién manda el precio” para evitar diferencias.
